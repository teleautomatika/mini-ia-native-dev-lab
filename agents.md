## Objetivo do projeto
Este projeto é um mini laboratório local full stack para fins educacionais.
## O que este projeto deve priorizar
- Simplicidade
- Clareza
- Organização
- Mudanças pequenas e fáceis de revisar
## Regras para agentes de IA
1. Não adicionar bibliotecas sem necessidade real.
2. Não transformar o projeto em algo complexo.
3. Sempre explicar antes de sugerir mudanças grandes.
4. Priorizar legibilidade sobre sofisticação.
5. Preservar a estrutura atual de pastas.
6. Nunca inventar credenciais, chaves ou segredos como se fossem reais.
7. Quando sugerir alteração, lembrar o usuário de testar:
 - docker compose up --build
 - http://localhost:8080
 - http://localhost:8000/projects
 - http://localhost:8000/docs
8. Sugerir mudanças compatíveis com iniciantes.
